Setup Instructions

  1. Create a GitHub repository (e.g. you/snap-tests) for your test suites.
  2. Download the workflow template above and commit it to .github/workflows/snap-test.yml.
  3. Add a repository secret named SNAP_DASHBOARD_GITHUB_TOKEN with a token that has repo scope.
  4. Create YARF test suites under suites/<snap_name>/suite/__init__.robot.
  5. Add a label named snap-test-results to the repository.
  6. Enter the repository name above and save.
